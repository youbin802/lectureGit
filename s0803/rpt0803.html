    웹 페이지, 외부 사이트 연결

    a 태그 속성

  • _blank : 새로운 탭 or 창
  • _self : 현재 탭 or 창
  • _parent : 화면 불러낸 부모 탭 or 창
  • _top : 최상위 탭 or 창

    input 태그 속성

  • readonly : 읽기 전용 필드
  • max/min : 최대값/최소값 지정
  • maxLength : 최대 문자 개수 지정
  • required : 필수 입력 필드 지정(빈칸이면 안 넘어감)
